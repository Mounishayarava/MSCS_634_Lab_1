# MSCS 634 - Lab 1

## Data Visualization, Preprocessing, and Statistical Analysis

### Overview
This lab focuses on analyzing a sales dataset using Python in Jupyter Notebook. The main goal is to understand data patterns through visualization, preprocessing, and statistical analysis.

### Steps Performed
- Data collection using Pandas
- Data visualization using Matplotlib
- Handling missing values
- Outlier detection using IQR
- Data scaling using MinMaxScaler
- Statistical analysis (mean, median, variance)

### Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

### Key Insights
- Sales and quantity show a positive relationship
- Most sales fall in mid-range values
- Scaling helps normalize the dataset
